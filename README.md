## quit and save
quit without save
```
esc -> type :q!
```

quit and save
```
esc -> type :wq
```

save without quit
```
esc -> type :w
```

## navigation

press `h`, `j`, `k`, or `l` in normal mode.
    - `h`: left
    - `j`: down
    - `k`: up
    - `l`: left


## typing
delete one character
```
x
```

add word after cursor
```
a
```

## deletion
Delete a word
```
dw
```

delete a line
```
dd
```

delete multiple line
```
$dd

// $ means number, e.g 3dd (delete 3 lines)
```

## backup
undo
```
u
```

redo
```
ctrl+r
```
